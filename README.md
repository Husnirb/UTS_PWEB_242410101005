Website yang saya buat adalah PawMeal yang mana merupakan dashboard jadwal makan kucing yang dirancang untuk pemilik kucing rumahan agar dapat memantau pemberian nutrisi anabul secara teratur. tujuan dikembangkannya untuk meminimalisir lupa atau double feeding. Website ini menyediakan pencatatan waktu, porsi, status makan, dan mood tracker kucing mereka.
